# Gaspard Kirira

Official profile page of **Gaspard Kirira**, founder of **Softadastra** and creator of **Vix.cpp**.

This repository contains the static website served at:

```txt
https://gaspardkirira.softadastra.com
```

## Purpose

This site is the canonical public profile for Gaspard Kirira. It connects the work around Softadastra, Vix.cpp, Softadastra Cloud, Rix, and related tools into one clear identity page.

The site is intentionally simple. It is built as static HTML, CSS, and JavaScript so it remains fast, readable, easy to host, and easy for search engines to understand.

## Local preview

Run a local server from the project root:

```bash
python3 -m http.server 8080
```

Then open:

```txt
http://localhost:8080
```

## Deployment

The site is designed to be deployed with GitHub Pages using the custom domain:

```txt
gaspardkirira.softadastra.com
```

The `CNAME` file must contain:

```txt
gaspardkirira.softadastra.com
```
